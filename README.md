# Welcome to the OpenBirding React Native app

The app provides an intuitive way to explore birding hotspots on your mobile device.

- [Download the app for iOS](https://apps.apple.com/us/app/openbirding/id6755897167)
- View more details at https://openbirding.org
- [Explore the backend API](https://github.com/rawcomposition/openbirding).

Feel free to create an issue for any bugs you encounter, or reach out adam@openbirding.org if you have questions or would like to contribute.
